HTML

HTML é uma linguagem de marcação de hipertexto, que é utilizada para criar páginas web. Ela é composta por tags, que são códigos que definem o que será exibido na página.

Em um arquivo html temos head e body, head se trata das configuracoes do arquivo, como o titulo, a codificacao, e body se trata do conteudo do arquivo, como textos, imagens, links, etc.

CSS

CSS é uma linguagem de estilização de páginas web. Com ela é possível definir cores, tamanhos, fontes, espaçamentos, entre outros.

Para utilizar o CSS, é possível inserir o código diretamente no arquivo HTML, ou criar um arquivo separado com a extensão .css e importá-lo no arquivo HTML.

JavaScript

JavaScript é uma linguagem de programação que permite adicionar interatividade às páginas web. Com ela, é possível criar animações, validar formulários, entre outras funcionalidades.

Assim como o CSS, o JavaScript pode ser inserido diretamente no arquivo HTML ou em um arquivo separado com a extensão .js.
