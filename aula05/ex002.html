Variáveis

Estudo de Variáveis

Uma variável é um espaço na memória do computador que armazena um valor. Esse valor pode ser um número, um texto, um objeto, entre outros.

Para criar uma variável em JavaScript, utilizamos a palavra-chave var, seguida do nome da variável e do valor que ela irá armazenar.

Exemplo:

var numero = 10;

var texto = 'Olá, mundo!'

Em Js temos como tipos primitivos de dados: string, number, boolean, null, undefined, symbol. E como tipos de dados compostos: object, array, function.

O comando typeof é utilizado para verificar o tipo de uma variável. Exemplo: typeof numero; // retorna 'number'
