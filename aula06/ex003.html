Tratamento de Dados

Nessa aula será vista a manipulação de dados utilizando prompt

O sinal de + no contexto de JavaScript pode ser tanto utilizado para soma quanto para concatenação de strings. No caso de soma, ele irá somar os valores das variáveis, e no caso de concatenação, ele irá juntar os valores das variáveis. Para forçar a soma de números, é necessário converter os valores para números antes de realizar a operação, fazendo uso da função Number().

Nota: O comando window.prompt() sempre retorna uma string

Exemplo:

        
            var n1 = prompt('Digite o primeiro número');
            var n2 = prompt('Digite o segundo número');
            var s = n1 + n2; // Aqui s será uma string
            var s = Number(n1) + Number(n2); // Aqui s será um número

Outras maneiras de realizar essa conversão é com os comandos Number.parseInt() e Number.parseFloat(), que convertem para inteiros e decimais, respectivamente.

De maneira análoga, para converter um número para string, é possível utilizar o comando String() ou o método toString().

Algumas outras funções notáveis para formatação numérica são toFixed(), que formata um número com uma quantidade específica de casas decimais, e toLocaleString(), que formata um número de acordo com a localidade do usuário. Outro método para mudar um possível ponto por vírgula é o replace(), que substitui um caractere por outro.

Exemplo:

        
            var num = 123.456;
            var numStr = num.toString();
            var numStr2 = String(num);
            var numFix = num.toFixed(2);
            var numLoc = num.toLocaleString('pt-BR', {style: 'currency', currency: 'BRL'});
            var numVirg = numLoc.replace('.', ',');

Formatação de Strings

Para inserir uma variável dentro de uma string é possível utilizar a concatenação, porém, uma maneira mais prática é utilizando template strings, que são strings delimitadas por crases, e que permitem a inserção de variáveis utilizando ${}.

Exemplo:

        
            var nome = 'João';
            var idade = 20;
            var frase = `O ${nome} tem ${idade} anos`;

Alguns outros métodos úteis para manipulação de strings são: length, que retorna o tamanho da string, toUpperCase(), que converte a string para maiúsculas, e toLowerCase(), que converte a string para minúsculas.

A função document.write() é utilizada para escrever conteúdo na página, e pode ser utilizada para escrever variáveis, strings, e até mesmo códigos HTML. Porém, seu uso não é recomendado, pois sobrescreve todo o conteúdo da página, e pode causar problemas de compatibilidade. O ideal é utilizar o método innerHTML do objeto document, que permite inserir conteúdo HTML em um elemento específico da página. Outra maneira de inserir conteúdo em um elemento é utilizando o método innerText, que insere apenas texto, sem interpretar tags HTML. Além disso, document.writeln() é utilizado para escrever conteúdo na página quebrando linhas. A tag br/ também é utilizada para quebra de linhas no HTML.
