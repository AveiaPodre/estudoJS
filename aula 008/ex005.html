Operadores(Continuação)

Essa aula é a continuação da última aula de operadores em JavaScript

Os operadores em JavaScript são divididos em várias categorias, como aritméticos, de atribuição, de comparação, lógicos, de incremento e decremento, de concatenação, entre outros.

Hoje veremos os operadores relacionais, lógicos e ternários

Operadores Relacionais

Os operadores relacionais são usados para comparar dois valores. Eles retornam um valor booleano, ou seja, verdadeiro ou falso.

Os operadores relacionais são:

  • == (igual a)
  • != (diferente de)
  • === (igual a e do mesmo tipo)
  • !== (diferente de e de tipos diferentes)
  • > (maior que)
  • < (menor que)
  • >= (maior ou igual a)
  • <= (menor ou igual a)

Operadores Lógicos

Os operadores lógicos são usados para combinar duas ou mais expressões condicionais. Eles retornam um valor booleano, ou seja, verdadeiro ou falso.

Os operadores lógicos são:

  • && (e)
  • || (ou)
  • ! (negação)

A ordem de precedência dos operadores lógicos é a seguinte: !, &&, ||. Ou seja, primeiro é feita a negação, depois a conjunção e, por último, a disjunção.

Operador Ternário

O operador ternário é um operador condicional que pode ser usado para substituir a estrutura if-else. Ele é composto por três partes: uma expressão condicional, uma expressão que é avaliada se a condição for verdadeira e uma expressão que é avaliada se a condição for falsa.

Sua sintaxe é a seguinte:

condição ? expressão1 : expressão2

Se a condição for verdadeira, a expressão1 é avaliada. Se a condição for falsa, a expressão2 é avaliada.

Exemplo:

         
            var idade = 18;
            var status = (idade >= 18) ? "Maior de Idade" : "Menor de Idade";
            console.log(status); //Maior de Idade
            //Se a idade for maior ou igual a 18, a variável status receberá o valor "Maior de Idade". Caso contrário, ela
            //receberá o valor "Menor de Idade".
