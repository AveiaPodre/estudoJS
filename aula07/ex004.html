Operadores

Nessa aula serão vistos alguns operadores em JavaScript

Os operadores em JavaScript são divididos em várias categorias, como aritméticos, de atribuição, de comparação, lógicos, de incremento e decremento, de concatenação, entre outros.

Hoje veremos apenas os operadores aritméticos e de atribuição

Operadores Aritméticos

Os operadores aritméticos são utilizados para realizar operações matemáticas, como soma, subtração, multiplicação, divisão, resto da divisão, e exponenciação.

Exemplo:

        
            var n1 = 10;
            var n2 = 5;
            var s = n1 + n2; //soma s = 15
            var s = n1 - n2; //subtração s = 5
            var s = n1 * n2; //multiplicação s = 50
            var s = n1 / n2; //divisão s = 2
            var s = n1 % n2; //resto da divisão inteira s = 0
            var s = n1 ** n2; //potenciação s = 100000

Outros operadores aritméticos são o ++ e o --, que incrementam e decrementam uma variável, respectivamente.

Exemplo:

        
            var n = 10;
            n++; //n = 11
            n--; //n = 10

A ordem de precedência dos operadores aritméticos é a mesma da matemática, sendo a exponenciação a de maior precedência, seguida da multiplicação e divisão, da soma e subtração, e por último o resto da divisão.

Operadores de Atribuição

Os operadores de atribuição são utilizados para atribuir valores a variáveis. O operador de atribuição mais comum é o =, que atribui um valor a uma variável.

Exemplo:

        
            var n = 10
            var s = 'Olá'
            var d = 9/n

Outros operadores de atribuição são o +=, -=, *=, /=, %=, **=, que realizam a operação aritmética e atribuem o resultado à variável.

Exemplo:

        
            var n = 10
            n += 5 //n = 15
            n -= 5 //n = 10
            n *= 5 //n = 50
            n /= 5 //n = 10
            n %= 5 //n = 0
            n **= 2 //n = 100
